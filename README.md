# Text Sentence Sorter (C++ & ASM) 

Утилита для анализа и сортировки предложений в текстовых файлах по количеству знаков. Проект демонстрирует низкоуровневую оптимизацию алгоритмов на C++ с использованием вставок **Assembly (MASM)**.

> **Автор:** Близученко Андрей (ИВТ2-23)
> **Статус:** Курсовая работа (защищена)

##  Задача
Разработать программу, способную:
1. Считывать текст из файла произвольного размера.
2. Разбивать текст на предложения.
3. Сортировать предложения по возрастанию/убыванию количества символов.
4. Использовать ассемблерные вставки для повышения производительности операций.

##  Техническая реализация
Проект выполнен в **Visual Studio 2022** и включает:
*   `main.cpp` — точка входа и высокоуровневая логика.
*   `List.h` — **собственная реализация** структуры данных (связный список) для хранения предложений. Это позволяет гибко управлять памятью, избегая оверхеда стандартных контейнеров `std::vector` или `std::list`.
*   **Inline ASM** — ключевые операции сравнения и перемещения данных написаны на ассемблере для прямого доступа к регистрам CPU.

##  Ключевые особенности
*   ✅ **Работа без STL:** Полностью ручное управление памятью и структурами данных.
*   ✅ **Гибридный код:** Совмещение C++ для IO-операций и ASM для вычислительной логики.
*   ✅ **Алгоритмическая база:** Реализация эффективных алгоритмов парсинга текста на предложения.

##  Как запустить
1. Откройте `TextFileSorter.sln` в Visual Studio 2022.
2. Убедитесь, что выбрана платформа **x86** (требуется для корректной работы `__asm` блоков).
3. Скомпилируйте и запустите проект.
4. Программа попросит указать путь к текстовому файлу для обработки.
